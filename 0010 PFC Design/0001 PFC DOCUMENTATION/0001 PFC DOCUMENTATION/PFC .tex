\subsection{AC Filter}
At the AC input, we can experience 2 Types of EMI (Electromagnetic interference) noise \ref{fig:ac_filter}: Differential Mode (DM) and Common Mode (CM) Noise. DM noise is conducted on the line and neutral, in opposite directions. The basic DM filter uses a single-winding inductor inserted into the line path, along with a capacitor from line to neutral, thus blocking noise from propagating through the system. CM noise is noise conducted on both the line and neutral (ground) but in the same direction. The basic CM filter uses a dual-winding inductor in both line and neutral paths, plus a capacitor from line to ground.
